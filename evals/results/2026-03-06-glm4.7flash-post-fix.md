## Analysis

The OS detection fix had the biggest impact — battery went from 5→1 steps,
hostname from 2→1, and open-app from broken to working. These are pure
system prompt improvements that help any model.

The NL false positive gating fixed the "show me processes" misfire without
breaking legitimate file tasks. The keyword list is deliberately conservative
(~40 words) to avoid over-filtering.

The remaining failure (wifi) is model knowledge — glm-4.7-flash doesn't know
`networksetup -getairportnetwork en0` well enough to try it early. This could
be improved by adding a few macOS command examples to the system prompt, but
that risks over-fitting to specific eval tasks. A better fix would be a larger
model or fine-tuning.
